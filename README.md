# 🍴 QuickEats

QuickEats is a modern food delivery website where users can browse food, filter by category, check ratings, add to cart, and place orders. It includes real authentication, payment integration, and an admin panel for managing orders.

---

## 🚀 Features
- 🔍 Search and filter food items  
- 🥗 Veg/Non-Veg toggle  
- ⭐ Ratings and reviews  
- 🛒 Cart with quantity update  
- 📜 Order history tracking  
- 🔑 Firebase authentication (Login/Signup/Logout)  
- 💳 Razorpay integration for payments  
- 📍 Location-based delivery availability  
- 📊 Admin panel to manage orders
